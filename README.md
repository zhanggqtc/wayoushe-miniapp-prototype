# 高校瓦友社小程序 H5 交互稿

这是无畏契约高校运营方向的小程序交互原型，用于对外展示和设计团队后续深化。

## 在线预览

GitHub Pages 发布后访问：

```text
https://zhanggqtc.github.io/wayoushe-miniapp-prototype/
```

## 文件说明

- `index.html`：可点击 H5 交互原型。
- `interaction-spec.md`：交互说明书，包含信息架构、核心模块、状态规则和设计团队关注点。

## 当前产品范围

- 高校瓦友社首页。
- 高校特权皮肤随机与下月自选槽。
- 活动与比赛中心。
- 高校先锋活动组织链路。
- 活动积分与小程序内积分商城入口。
- 高校同好雷达。
- 我的页个人资产与记录。

## 设计重点

1. 首页要突出高校归属感、皮肤权益和活动雷达。
2. 皮肤自选必须区分“已占位”“待锁定”“已锁定”“默认补位”。
3. 活动卡需要提供足够决策信息，包括报名、地点、人数、积分和状态。
4. 积分商城在小程序内完成商品浏览、详情查看和兑换确认，确认后跳转外部成熟周边商城承接履约。
5. 高校先锋能力不作为底部 Tab，只在身份解锁后展示。
